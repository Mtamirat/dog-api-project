## Description
This project uses the Fetch API to retrieve data from the Dog API.
It displays:
- List of dog breeds
- Detailed breed information
- Random dog facts
- Dog groups

## Technologies Used
- HTML
- JavaScript
- Fetch API
- Dog API

## How to Run
1. Clone the repository
2. Open in VS Code
3. Install Live Server extension
4. Right-click index.html
5. Click "Open with Live Server"

## Features
- API GET requests
- JSON parsing
- Error handling
- Interactive breed selection
